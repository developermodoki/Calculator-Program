# Calculator-Program
Pythonを使った計算プログラムファイルです。

# 使い方
まず、足し算か引き算かどちらか選ぶよう表示されたら、1か2のどちらかを入力してください。（1が足し算、2が引き算です。）

# 足し算の使い方
まず、足される数を入力するよう求めたれたら、足される数を入力してください。そしたら、次に足す数を入力するよう求められます。なので、求められたら、足す数を入力してください。
すると、答えが表示されます。

# 引き算の使い方
まず、引かれる数を入力するよう求められます。なので、引かれる数を入力してください。そして、次に引く数を入力するよう求められます。なので、引く数を入力してください。すると、
答えが表示されます。

# 注意事項
このファイルを実行するには、Python3.8の環境が必要です。Python3.8の環境を準備していただくようお願いします。
